# EarlyBird is an offer, not a plan

EarlyBird changes the commercial price but does not change product access, so it will be modeled as an `Offer` applied to a real `Plan` instead of as a standalone plan. This keeps Starter, Growth, and Scale as stable packages while allowing EarlyBird limits, locked pricing, and redemption counts to evolve without infecting permissions, invoices, renewals, and dashboard billing.
